# Smart Calculator (Python CLI)

A simple **command-line calculator** built in Python.  
Supports basic arithmetic operations with proper **input validation** and **continuous calculation**.

---

## Features

- Addition, Subtraction, Multiplication, Division
- Handles invalid input (letters, symbols)
- Prevents division by zero
- Continuous calculation until user chooses to exit
- Uses Python functions and clean code practices
