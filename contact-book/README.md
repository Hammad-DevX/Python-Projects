# Contact Book (Python CLI)

A simple **command-line contact management system** built with Python.  
Allows users to store, search, view, and delete contacts using a clean menu-driven interface.

---

## Features

- Add new contacts with validation
- Prevent duplicate names and phone numbers
- View contacts in a formatted table
- Case-insensitive contact search
- Safe deletion of contacts
- User-friendly menu system
